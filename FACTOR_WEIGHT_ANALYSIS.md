## 🎯 權重合理性分析

### ✅ 合理的設計

1. **時間(16.7%)與雲層(16.7%)並列第一**
   - 這兩者確實是燒天最關鍵因素
   - 時間決定是否有燒天機會
   - 雲層決定燒天品質

2. **濕度(13.3%)排第三**
   - 濕度直接影響雲層色彩
   - 適當濕度讓雲層更容易呈現紅橙色
   - 權重設置合理

3. **次要因素分配平衡**
   - 溫度、能見度、風速、空氣品質各佔10%
   - 這些因素對燒天有影響但不是決定性的
   - 平衡分配避免任一因素過度影響

4. **低權重因素**
   - 氣壓(6.7%)和UV(6.7%)最低
   - 這些因素影響相對較小
   - 不應佔太高比重

## ⚠️ 潛在問題

### ❌ 問題 1: 時間因子權重過高 (25分/16.7%)

**問題分析:**
- 時間因子主要判斷「是否在燒天時段」
- 這是**門檻條件**而非**品質因素**
- 時間對了但天氣不好，燒天機率仍然很低
- 目前設計讓時間因子過度影響最終分數

**實際影響:**
```
例子：17:30（黃金時段） + 厚雲密布 + 能見度差
- 時間因子: 25分（滿分）
- 雲層因子: 5分（厚雲）
- 能見度: 3分（很差）
- 總分可能達到 50-60分 → 系統判斷為「中等」燒天機率
- 實際情況：幾乎不可能有燒天
```

**建議調整:**
- 降低到 **15-18分 (10-12%)**
- 改為「資格判斷」而非主要評分標準
- 時間不對直接扣分，時間對了給基礎分即可

---

### ❌ 問題 2: 雲層因子權重不足 (25分/16.7%)

**問題分析:**
- 雲層是燒天的**必要條件**
- 沒有適當雲層，其他條件再好也不會有燒天
- 目前權重與時間因子相同，但重要性應該更高

**實際影響:**
```
例子：18:00（最佳時間） + 萬里無雲 + 完美能見度
- 時間因子: 25分
- 雲層因子: 2分（無雲）
- 其他因子: 50分（良好）
- 總分可能達到 75分 → 系統判斷「良好」燒天
- 實際情況：完全沒有燒天（無雲層）
```

**建議調整:**
- 提升到 **30-35分 (20-23%)**
- 雲層應該是權重最高的單一因素
- 無雲或過厚雲層應該大幅降低總分

---

### ❌ 問題 3: 能見度權重可能過低 (15分/10%)

**問題分析:**
- 能見度直接影響燒天的清晰度和色彩飽和度
- 霧霾會大幅降低燒天品質
- 即使有雲層，能見度差會讓燒天變得灰濛濛

**實際影響:**
```
例子：完美時間 + 適量雲層 + 嚴重霧霾
- 能見度只佔10%，影響有限
- 但實際上會導致燒天品質大幅下降
- 顏色不鮮豔、層次感差
```

**建議調整:**
- 提升到 **20分 (13%)**
- 與濕度並列第三重要因素
- 能見度差應該顯著降低評分

---

### ❌ 問題 4: UV指數意義不大 (10分/6.7%)

**問題分析:**
- UV指數與燒天品質關聯性較低
- 主要影響是太陽輻射強度
- 在燒天預測中不是關鍵因素

**建議調整:**
- 降低到 **2-5分 (1-3%)**
- 或考慮完全移除
- 將分數重新分配給更關鍵因素

---

## 📈 建議的優化方案

### 優化後的權重分配

| 因子 | 當前 | 建議 | 變化 | 理由 |
|------|------|------|------|------|
| **雲層 (cloud)** | 25分 (16.7%) | **35分 (23%)** | **+10** | 最關鍵因素，應該權重最高 |
| **濕度 (humidity)** | 20分 (13.3%) | **20分 (13%)** | 不變 | 權重合理 |
| **能見度 (visibility)** | 15分 (10%) | **20分 (13%)** | **+5** | 影響燒天清晰度 |
| **時間 (time)** | 25分 (16.7%) | **18分 (12%)** | **-7** | 改為門檻判斷 |
| **溫度 (temperature)** | 15分 (10%) | **15分 (10%)** | 不變 | 權重合理 |
| **風速 (wind)** | 15分 (10%) | **15分 (10%)** | 不變 | 權重合理 |
| **空氣品質 (air_quality)** | 15分 (10%) | **15分 (10%)** | 不變 | 權重合理 |
| **氣壓 (pressure)** | 10分 (6.7%) | **10分 (7%)** | 不變 | 權重合理 |
| **UV指數 (uv)** | 10分 (6.7%) | **2分 (1%)** | **-8** | 關聯性低 |
| **總分** | **150分** | **150分** | - | 維持不變 |

### 調整理由總結

1. **強化雲層因子** (+10分)
   - 雲層是燒天的必要條件
   - 應該是權重最高的單一因素
   - 能更準確識別「無雲」情況

2. **提升能見度權重** (+5分)
   - 能見度直接影響燒天品質
   - 霧霾會顯著降低燒天效果
   - 與濕度並列為重要因素

3. **降低時間因子** (-7分)
   - 時間改為「門檻判斷」
   - 不應過度影響最終評分
   - 時間對了給基礎分，時間不對扣分

4. **減少UV影響** (-8分)
   - UV指數與燒天品質關聯性低
   - 分數應分配給更關鍵因素
   - 保留少量分數以保持完整性

5. **其他因素維持平衡**
   - 溫度、風速、空氣品質、氣壓權重合理
   - 不需要調整

## 🎯 預期效果

### 改善項目

1. **降低誤報率**
   - 更準確識別有雲層但能見度差的情況
   - 晴空無雲時不會給出高分
   - 時間對但天氣差時不會過度樂觀

2. **提升準確率**
   - 雲層因素得到更高權重
   - 能見度影響被正確反映
   - 整體預測準確率預計提升 **5-8%**

3. **更符合實際經驗**
   - 有雲層才有燒天（雲層權重提升）
   - 能見度差會降低品質（能見度權重提升）
   - 時間只是基本條件（時間權重降低）
