### 技術特性

✨ **深色玻璃態設計**
- 背景: 漸層 rgba(255,255,255,0.12) → rgba(255,255,255,0.08)
- 模糊效果: backdrop-filter: blur(10px)
- 陰影: 0 8px 25px rgba(0,0,0,0.1)
- 邊框: 1px solid rgba(255,255,255,0.2)

📱 **響應式佈局**
- 使用 CSS Grid: `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))`
- 間距: gap: 12px
- 自適應不同屏幕寬度

🎯 **導航卡片設計**
- 圖標 + 文字組合
- 懸停效果準備好 (可在 CSS 中追加)
- 一致的內邊距和邊框半徑
